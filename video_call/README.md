# video_call

## 📃 Project Info
> 개발 환경: Flutter SDK 3.3.x
>
> 플러그인: agora_rtc_engine: 6.0.0, permission_handler: 10.0.2

***

## ⚙️ 기능
- 실제 핸드폰을 이용해 1:1 화상 통화 실행/종료

***

## 🛠️ 조작법
- [입장하기] 버튼을 눌러 화상통화에 참여하기
- [채널 나가기] 버튼을 눌러 화상통화 종료하기

***

## 💡 메인 구성 요소
- 아고라 API
- 권한 관리
- WebRTX
